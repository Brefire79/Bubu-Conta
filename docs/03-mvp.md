# 03 — MVP

## Escopo do MVP

### Deve ter
- Login via Supabase Auth.
- Dashboard mensal.
- Cadastro de contas recorrentes.
- Geração das contas do mês.
- Status pago/a pagar/atrasado.
- Upload de comprovante no Supabase Storage.
- Histórico mensal.
- Relatório simples por mês.
- Responsivo mobile.

### Pode ter na fase 2
- IA OCR para ler comprovante.
- Envio automático para WhatsApp.
- Convite de familiar.
- Notificações push.
- Previsão de gastos.

### Não fazer no MVP
- Open Finance.
- Pagamento Pix dentro do app.
- ERP completo.
- Controle bancário avançado.
- Integração WhatsApp não oficial.

## Métrica de sucesso
Usuário consegue cadastrar suas contas fixas e controlar um mês inteiro em menos de 5 minutos.
