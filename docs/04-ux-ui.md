# 04 — UX/UI

## Referência visual
Usar a imagem `assets/tela-referencia-app-bubu.png` como base principal.

## Estilo visual
- Tema escuro azul-marinho.
- Bordas amarelas para destaque do resumo mensal.
- Cards vermelhos para contas a pagar.
- Cards verdes para contas pagas.
- Tipografia grande e legível.
- Ícones circulares.
- Layout mobile-first.

## Cores sugeridas
- Fundo principal: `#001A33` ou similar.
- Azul profundo: `#002B55`.
- Amarelo destaque: `#FFD22E`.
- Vermelho pendente: `#FF3131`.
- Verde pago: `#2ECC40`.
- Branco texto: `#FFFFFF`.
- Cinza secundário: `#B8C2CC`.

## Tela inicial
Elementos obrigatórios:

1. Header
   - menu hambúrguer;
   - título APP Bubu;
   - sino de notificações.

2. Card mês referência
   - botão voltar mês;
   - mês/ano grande;
   - botão avançar mês;
   - total a pagar;
   - total pago;
   - total do mês.

3. Lista de contas
   - cards verticais grandes;
   - status colorido;
   - valor destacado;
   - vencimento visível;
   - seta para detalhe.

4. Navegação inferior
   - Início;
   - Contas;
   - Relatórios;
   - Configurações.

## Comportamento visual
- Conta paga fica verde.
- Conta pendente fica vermelha.
- Conta atrasada pode ficar vermelho intenso com etiqueta `ATRASADO`.
- Ao marcar como paga, card muda de vermelho para verde com animação leve.
